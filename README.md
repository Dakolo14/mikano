# mikano

Mikano International Limited — Konga Store-in-Store storefront.

## Pages

- **index.html** — Hub landing (choose Mikano Motors or Mikano Power)
- **motor.html** — Vehicles (Changan, Deepal, Maxus)
- **power.html** — Generators and power systems

## Local preview

Open `index.html` in a browser, or run a simple static server:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`
